Tiny Brain Wallet

Don’t trust. Verify.

An minimalistic and easy-to-audit crypto HD wallet designed around minimal dependencies and a simple, verifiable architecture.

Goal

Keep the dependency tree small and the architecture simple so security auditors and developers can quickly understand and verify the wallet’s behavior.

Minimal dependencies

  • bitcoinjs-lib — Bitcoin protocol implementation
  • tiny-secp256k1 — minimal secp256k1 primitives
  • ecpair — elliptic curve key pair management
  • bip32 — BIP32 key derivation
  • bip39 — mnemonic generation / seed derivation
  • electron — desktop application framework
  • react + react-dom — UI rendering
  • react-router — routing

Everything else is implemented in-house to keep behavior transparent.

Currently supported cryptocurrencies

Bitcoin

  • API provider: mempool.space
  • Address type: Native SegWit (P2WPKH)
  • UTXO: discovery + selection

⏳ More cryptocurrencies coming soon...
